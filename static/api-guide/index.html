Gemini Vector Search API 指南

欢迎使用 Gemini Vector Search API，这是一个基于 Google Gemini 2.0 Flash 模型的 API 系统。

主要功能

  • 向量检索：将文档转换为向量并执行相似度搜索
  • 文档管理：上传、检索和管理文档
  • 智能问答：基于文档内容回答问题
